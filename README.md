# 🌐 Personal Portfolio Website

Welcome to my personal portfolio website!
This project showcases my skills, projects, experience, and achievements as a developer.

## 🚀 Features

* Responsive modern UI
* About Me section
* Skills showcase
* Projects gallery
* Resume download
* Contact form
* Social media links
* Smooth animations and transitions

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Git & GitHub

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Aniket6- Prog/portfolio-website.git
```

Navigate to the project directory:

```bash
cd portfolio-website
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```
